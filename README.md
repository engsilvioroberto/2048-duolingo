# 2048 Duolingo 🎮

Um joguinho 2048 estilo **Duolingo** — mobile-first, verde vibrante, React.

[![Deploy to GitHub Pages](https://github.com/engsilvioroberto/2048-duolingo/actions/workflows/deploy.yml/badge.svg)](https://github.com/engsilvioroberto/2048-duolingo/actions/workflows/deploy.yml)

## Como jogar

- **Desktop**: setas do teclado
- **Mobile**: deslize os dedos
- Junte números iguais para somá-los
- Chegue no **2048** para vencer!

## Stack

- React 18 + Vite
- Inter (Google Fonts)
- CSS puro — estilo Duolingo

## Desenvolvimento

```bash
npm install
npm run dev
```

## Deploy

Push na `main` → GitHub Actions faz deploy automático no GitHub Pages.
